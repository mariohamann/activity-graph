<activity-graph>

The <activity-graph> web component visualizes activity data. It is a simple way to show activity over time as known from GitHub's contribution graph. It is built with accessibility, performance, progressive enhancement and flexibility in mind.

Attributes

Range Start/End

Use the range-start and range-end attributes to define the range of the graph. <activity-graph range-start="2024-01-01" range-end="2024-01-31"></activity-graph> <activity-graph range-start="2024-01-01" range-end="2024-12-31"></activity-graph>

Activity Data

The activity-data attribute expects a comma-separated list of dates. By setting dates multiple times the activity level can be increased.

<activity-graph activity-data="2024-01-01,2024-01-01,2024-01-01,2024-01-04,2024-01-05" ></activity-graph>

As an alternative you can utilize progressive enhancement to fill the data by setting children

Activity Levels

The activity-levels attribute expects a comma-separated list of numbers. The numbers define the threshold for each level. The first number is the lowest level, the last number is the highest level. The number of levels is the length of the list. The default is 0,1,2,3.

The following example shows a graph with three levels, where the second level has a threshold of 2 and the third level has a threshold of 4.

<activity-graph activity-data="2024-01-01,2024-01-01,2024-01-01,2024-01-04,2024-01-05" activity-levels="0,2,4" ></activity-graph>

First Day of Week

Per default the first day of week is set to 0 (Sunday). Set a number between 0 and 6, representing the day of the week, to alter the day. 

<activity-graph first-day-of-week="1"></activity-graph>

i18n

Use the lang attribute to set the language of the graph. This will set the language of the weekdays, months etc.

Use the i18n attribute to customize the labels. The default is set to {"activities":"Activities","less":"Less","more":"More"}.

<activity-graph i18n='{"activities":"Aktivitäten","less":"Weniger","more":"Mehr"}' lang="de" ></activity-graph>
